Model Building¶

Let's first build a baseline model. The idea behind it is to include only a few features from our dataset to see how well it performs on the test set. Once we get the baseline model accuracy, we will compare it to an advanced model that includes many other features - most of which are generated using feature engineering.

Baseline Model¶

# define the feature set for our baseline model
predictors = ['ATT_diff', 'MID_diff', 'DEF_diff','rank_pts_diff']

# scale the features
def scale_df(df, means, stds):
    cols_to_scale = means.index
    df = df.copy()
    df[cols_to_scale] = (df[cols_to_scale] - means) / stds
    return df

cols_to_scale = predictors

train_means = train_data[cols_to_scale].mean(axis=0)
train_stds = train_data[cols_to_scale].std(axis=0)

Decision Tree¶
In [8]:
# define the list of depths we want to test
depths = list(range(1, 31))
train_scores = []
cvmeans = []
cvstds = []

for depth in depths:
    dtree = DecisionTreeClassifier(criterion='gini', max_depth=depth)
    dtree.fit(x_train, y_train)
    
    # Perform 5-fold cross validation and store results
    train_scores.append(dtree.score(x_train, y_train))
    scores = cross_val_score(dtree, x_train, y_train, cv=5)
    cvmeans.append(scores.mean())
    cvstds.append(scores.std())

cvmeans = np.array(cvmeans)
cvstds = np.array(cvstds)

Training accuracy hits nearly perfect at depth 26 but we don't think that a depth-26 tree will generalize well. DecisionTreeClassifier has some randomness in it, so depending on the random choices it made, depth 2, 3 or 4 trees might come out with better validation performance. We'll choose depth 3 here.

Train Set Accuracy at depth=3: 52.14%

For the above Decision Tree model, we choose a lower depth so that we don't generate an overfit tree. As a result though, this model underfits the training data. This drawback can be compensated by using Bagging, Random Forests and Boosting models which have less variance. So, we can use higher depths without worrying too much about overfitting on the train set.

# define the list of depths we want to test
depths = list(range(1, 31))
train_scores = []

    dtree = DecisionTreeClassifier(criterion='gini', max_depth=depth)
    dtree.fit(x_train, y_train)
    
    # Perform 5-fold cross validation and store results
    train_scores.append(dtree.score(x_train, y_train))
    scores = cross_val_score(estimator=dtree, X=x_train, y=y_train, cv=5)
    cvmeans.append(scores.mean())
    cvstds.append(scores.std())

cvstds = np.array(cvstds)
